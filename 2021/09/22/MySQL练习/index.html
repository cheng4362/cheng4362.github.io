# 三、操作表

题目如下：

1、查询所有的男生信息

2、查询每个班级的学生人数，显示班级名称，班级人数

3、统计每个地区的学生人数，并显示地区名称，人数

4、查询平均成绩大于 60 分的同学的学号和平均成绩；

5、查询每个学生的各科成绩 - 按学号进行升序排序，按科目成绩降序排序，结果显示学号，学生姓名，课程名称，课程分数

6、查询每个学生姓名和总成绩

7、查询学过 “Python 编程基础 “这门课的学生学号和姓名以及 Python 编程基础成绩

8、查询每个老师的 id，姓名和课程数量

9、查询有课程成绩小于 60 分的同学的学号、姓名；

10、查询各科成绩最高和最低的分：以如下形式显示：课程 ID，课程名称，最高分，最低分；

11、课程平均分从高到低显示；

12、查询每门课程被选修的学生数；
