# CSS 概念、作用和目的

# 概念

CSS 中文翻译为 “层叠样式表”，简称样式表，它是一种制作网页的新技术

# 作用

结构与表现相分离是指在网页设计中，HTML 标签只用于搭建网页的基本结构，不使用标签属性设置显示样式，所有的样式交由 CSS 来设置。

# 目的

让网页变好看，代码更清晰。

# CSS 的优点

1、样式表能实现内容与样式的分离 ，方便团队开发

2、样式复用、方便网站的后期维护

3、页面的精确控制，实现精美、复杂页
