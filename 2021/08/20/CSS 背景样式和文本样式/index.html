# CSS3 新增常用标签简介

​ CSS3 是 CSS 技术的升级版本，CSS3 语言开发是朝着模块化发展；网络浏览器也还将继续支持 CSS2。CSS3 主要的影响是将可以使用新的可用的选择器和属性。

・圆角边框：border-radius

・边框阴影：box-shadow

・背景图片的尺寸：background-size

・文本阴影：text-shadow
